# smm3ds-hd-chs

这是为在 3ds 模拟器（azahar）上玩《超级马里奥制造 3DS》（SMM3DS）准备的中文高清材质合集。本仓库只包含资源文件（纹理、拼音输入法、中文系统字体等），不包含游戏本体和游戏更新中文补丁。

**目录说明**

- `textures/`：游戏与界面的高清材质包（纹理贴图、UI 字体、关卡素材等）。
- `chinese_system_font/`：系统中文字体文件，可在游戏中显示完整中文。
- `pinyin_input/`：拼音输入法，用于在模拟器中使用拼音输入支持。

**使用方法（示例）**

1. 备份原始资源（始终先备份原文件）。
2. 将所需的 `textures/` 下的材质文件复制到模拟器的用户目录的 load 下面。
3. 替换系统字体，将 `chinese_system_font/` 中的字体文件（包含路径）复制到模拟器用户目录的 nand\00000000000000000000000000000000\ 下面，合并覆盖原文件。
4. 若需中文输入，将 `pinyin_input/` 中的输入法文件（包含路径）复制到模拟器用户目录的 nand\00000000000000000000000000000000\ 下面，合并覆盖原文件。

注意：如果不使用该系统中文字体，游戏中可能无法看到中文（如果没有安装中文字体），或者看到的中文不是高清化的（如果安装的是其它中文字体）。

中文字体和拼音输入法来自 3DS 系统汉化包，并已经做了解密处理。如果想要使用完整的汉化系统，可以自行通过某些渠道（这里也不方便提供）下载 3DS 系统完整汉化包并解密后替换。注意，如果使用的是 azahar 模拟器，不解密直接替换模拟器 nand 中的文件，会导致 azahar 无法正常使用，因为 azahar 只支持解密后的文件。

该高清材质包支持《超级马里奥制造 3DS》最新的 1.05 汉化补丁，打了补丁的话，是可以联网游戏的。但需要导入实机唯一数据，并且安装 nimbus，开启 pretendo。

**模拟器设置**

- 图形->增强性功能->渲染器->内部分辨率->8倍，或者你觉得比较合适的倍数，但如果选原生是没有效果的。

- 图形->增强性功能->使用自定义纹理。不启用该选项也看不到高清效果。

- 图形->高级选项->纹理采样->最邻近。不设置该选项会看到砖块和地面中间有裂缝。

**特征与限制**

- 中文字体高清化。
- UI 界面（比如按钮、背景、教学画面、菜单、图标等）高清化。
- MU 风格下，游戏角色高清化。

由于模拟器本身限制，MU 风格下面的金币、砖块、传送带等动画中的素材无法 dump 出来替换，所以这部分没有办法做高清化处理，MU 风格下的游戏场景如果做高清化，会导致上面提到的动画效果失效，尤其是传送带动画效果失效会严重影响游戏性，所以 MU 游戏场景没有做高清化处理。

总之，做到最后感觉很鸡肋，如果想玩高清化版本不如直接玩 WiiU 版本。但既然已经完成到这个程度了，还是打算分享出来，如果有朋友能够解决上面提到的问题，欢迎提交修改。
